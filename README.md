# Kidova Pro Landing Page

A premium product landing page built with **HTML, CSS, Tailwind CSS, JavaScript, GSAP, and ScrollTrigger**.

## Features

* Frame-by-frame scroll animation
* GSAP ScrollTrigger effects
* Fully responsive design (312px+)
* Smooth section reveal animations
* Premium modern UI
* Optimized performance

## Tech Stack

* HTML5
* CSS3
* Tailwind CSS
* JavaScript
* GSAP
* ScrollTrigger
